# Phase 5 Plan 01: Circuit Breaker & Performance

## Objective

Implement a diamond-level emergency pause mechanism and optimize gas-heavy loops in the withdrawal limiter and token transfer paths.

## Tasks

### T1: Add diamond-wide pause flag to GNUSControlStorage

**File:** `contracts/gnus-ai/GNUSControlStorage.sol`

Add `bool paused` to `Layout` struct (appended after `chainID` — safe, no storage displacement in diamond pattern):

```solidity
struct Layout {
    mapping(uint256 => mapping(address => bool)) bannedTransferors;
    mapping(address => bool) gBannedTransferors;
    uint256 bridgeFee;
    uint256 protocolVersion;
    uint256 chainID;
    bool paused;  // NEW
}
```

Add `whenNotPaused` modifier to the library:

```solidity
modifier whenNotPaused() {
    require(!layout().paused, "GNUSControl: contract paused");
    _;
}
```

### T2: Add pause/unpause to GNUSControl + wire into all mutative facets

**File:** `contracts/gnus-ai/GNUSControl.sol`

Add two functions guarded by `onlySuperAdminRole`:

```solidity
function pause() external onlySuperAdminRole {
    GNUSControlStorage.layout().paused = true;
}
function unpause() external onlySuperAdminRole {
    GNUSControlStorage.layout().paused = false;
}
```

**Wire into facets:** Replace `whenNotPaused` (OZ PausableUpgradeable) with `GNUSControlStorage.whenNotPaused()` on all state-changing functions across these facets:
- `GNUSERC1155MaxSupply._beforeTokenTransfer`
- `GNUSBridge`: `withdraw`, `bridgeOut`, `transfer`, `transferFrom`, `mint`, `burn`
- `ERC20TransferBatch`: `mintBatch`, `transferBatch`, `transferOrBurnBatch`
- `GNUSNFTFactory`: `pause`/`unpause` replaced with delegate to GNUSControl

### T3: Merge dual loops in _beforeTokenTransfer

**File:** `contracts/gnus-ai/GNUSERC1155MaxSupply.sol`

Current code has two sequential passes over `ids[]` — one aggregating `totalGNUSAmount`, one checking banned transferors. Merge into single pass:

```solidity
for (uint256 i = 0; i < ids.length; ++i) {
    if (ids[i] == GNUS_TOKEN_ID) {
        totalGNUSAmount += amounts[i];
    }
    if (GNUSControlStorage.isBannedTransferor(ids[i], operator)) {
        revert("GNUSNFTFactory: operator is banned");
    }
}
```
