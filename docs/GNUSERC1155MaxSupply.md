# GNUSERC1155MaxSupply

## Overview

#### License: MIT

```solidity
contract GNUSERC1155MaxSupply is ERC1155SupplyUpgradeable, PausableUpgradeable, ERC1155BurnableUpgradeable
```

This contract extends ERC1155 functionality with supply management, pausing, and burning capabilities.

This contract uses the GNUSNFTFactoryStorage and GNUSControlStorage libraries for additional storage management.
